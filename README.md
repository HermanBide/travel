# Travel-app

- React
- Nodejs
- MongoDB

Components 
- Navbar(Header)
- Registeration pages (screens) for login/registration
- Homepage (screens)
- Booking Page (screens)
- Rating pages (screens)
- banner (slider)
- Destination information
